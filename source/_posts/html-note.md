html 是超文本标记语言, 超文本就是图片音频等多媒体内容, 标记语言是一套标记标签, 如 <a>

html 的书写很不规范, xhtml 是 严格纯净的 html

输入 html:5 再按 tab 键生成内容

ASCII 码: 用 1 个字节表示字符

ANSI 码: 每个国家为了显示自己的语言, 都对 ASCII 码进行了扩展, 如中国的 ANSI 编码有 GB2312和 GBK. 用 2 个字节表示一个汉字

GBK: 对 GB2312 进行了扩展, 用来显示古汉语

Unicode 编码: 用 4 个字节表示一个字符

UTF-8: 根据字符的不同, 选择编码的长度, 如 A 用 1 个字节, 一个汉字用 2 个字节
